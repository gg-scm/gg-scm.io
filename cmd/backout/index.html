gg backout

gg backout [options] [-r] REV

reverse effect of an earlier commit

Prepare a new commit with the effect of REV undone in the current working copy. If no conflicts were encountered, it will be committed immediately (unless -n is passed).

Options

-e
-edit
invoke editor on commit message
-n
-no-commit
do not commit
-r rev
revision
