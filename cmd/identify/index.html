gg identify

gg identify [-r REV]

identify the working directory or specified revision

Print a summary of the revision or working directory if no revision was provided. The revision’s hash identifier is printed, followed by a “+” if the working copy is being summarized and there are uncommitted changes, a list of branches it is the tip of, and a list of tags.

Options

-r rev
identify the specified revision

Aliases

  • id
